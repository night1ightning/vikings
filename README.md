# Testwork Java 

## Task:

>Вам необходимо разработать программу, которая читает карту островов с маршрутами, создает N
>викингов и пришвартовывает их корабли к островам. Программа должна работать до тех пор, пока все викинги не
>будут уничтожены, либо застрянут на островах, либо не проплывут 10000 дней и у них кончится провизия.
>Также следует печатать в консоль если:
>1. Маяк на острове уничтожен: “АГР!!! На Остров1 уничтожен маяк, благодаря Викинг1 и Викинг2”
>2. Викинг застрял на острове: “АГР!!! Викинг1 застрял на Острове1 и больше не участвует в войне”
>После завершения программы, она должна распечатать все, что осталось от мира в том же формате, что и
>входной файл
>Создать 5 тестовых карт 

## Description
I create `Maven` project.
I use Thread for more performance
I make buffer of OUT stream for more performance.

## Run
1. Need install `jdk8` or above.
2. Execute console command in project directory ```mvn package```.
3. Down to target directory and copy jar any place.
4. Execute jar. ```java -jar vikings-1.0.0.jar```.
5. At the end of the output, will be path to "MAP"
